# README

This repository contains the circuits and experimental data and analysis scripts for experiments reported in the following paper:

Simon Martiel and Ali Javadi-Abhari, ``Low overhead error detection with spacetime codes'', arXiv:2504.15725.

The experiments were performed using Qiskit 1.2.1.
The scripts were tested with Python 3.12.5, numpy 2.2.0, scipy 1.14.1, matplotlib 3.9.2.
